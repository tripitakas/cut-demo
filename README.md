# cut-demo

基于 jQuery、Raphael 的切分校对单元测试和原型。

[在线预览](http://ggbstudy.top/cut/)

- [x] 页面缩放(1~5倍)
- [x] 显示字框和控制点，用不同颜色区分各种状态
- [x] 字框编辑：拖拽控制点
- [x] 字框编辑：移动和微调大小
- [x] 字框编辑：增加字框
- [x] 字框编辑：删除字框
- [ ] 字框跳转导航
- [x] 支持编辑和导航的快捷键
- [ ] 自动单元测试
- [ ] 缩放时当前字框位置保持不变
